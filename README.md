## 📌 Overview
This project is a UX/UI design simulation developed for a healthcare ecosystem inspired by HelixCare.

The goal is to design a modern, intuitive, and scalable digital platform that improves access to healthcare services and enhances user experience.

---

## 🎯 Objectives
- Create a clear and intuitive user journey  
- Build a strong and trustworthy healthcare interface  
- Improve accessibility and usability  
- Deliver a modern and scalable design system  

---

## 💡 Design Thinking
This project is based on a user-centered approach:

- **Clarity first** → Simplified navigation and structure  
- **Trust-driven design** → Clean layout adapted to healthcare  
- **Consistency** → Unified visual system (colors, typography)  
- **Accessibility** → Designed for diverse users and contexts  

---

## 🧩 Key Features
- Modern and clean interface  
- Structured content hierarchy  
- Intuitive navigation  
- Responsive design approach
